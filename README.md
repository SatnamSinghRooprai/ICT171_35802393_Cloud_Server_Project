# ICT171_35802393_Cloud_Server_Project
Cloud server project for ICT171 - Ubuntu EC2, Apache, DNS, SSL, Script
1. Introduction
CyberPath is a cloud‑hosted, interactive quiz website designed to help beginners identify which cybersecurity career path best matches their personality, interests, and problem‑solving style.
This project was developed as part of ICT171 – Introduction to Server Environments and Architectures, demonstrating cloud deployment, web development, documentation, and professional communication skills.

The website is fully deployed on an AWS EC2 Ubuntu server, accessible through a public domain, and secured with SSL/TLS.

2. Purpose of the Project
Many newcomers to cybersecurity struggle with one core question:

“Which cybersecurity field is right for me?”

Cybersecurity is broad — SOC analysis, penetration testing, cloud security, digital forensics, GRC, threat intelligence, and more.
Each field requires different traits, thinking styles, and strengths.

CyberPath solves this problem by offering:

A 10‑question personality and problem‑solving quiz

A tailored career recommendation

Suggested certifications for starting that career

Free learning resources to begin immediately

A modern, responsive UI built with HTML, CSS, and JavaScript

This makes cybersecurity career exploration simple, accessible, and personalized.

3. What the Website Does
CyberPath guides users through:

✔ A short interactive quiz
Questions explore how the user thinks, approaches problems, handles uncertainty, and prefers to work.

✔ A trait‑based scoring system
Each answer contributes to traits such as:

Vigilance

Curiosity

Precision

Aggression

Communication

Systems Thinking

✔ A career recommendation engine
Based on the trait profile, the website recommends one of several cybersecurity roles, such as:

SOC Analyst

Penetration Tester

Cloud Security Specialist

Digital Forensics Analyst

GRC Analyst

✔ Certification guidance
Each career path includes beginner‑friendly certifications such as:

CompTIA Security+

Microsoft SC‑200

Blue Team Level 1

Google Cybersecurity Certificate

✔ Free learning resources
The site provides links to:

TryHackMe

LetsDefend

CyberDefenders

Security Blue Team

Splunk Free

✔ A clean, modern UI
The interface is designed to be simple, fast, and visually appealing, making the quiz enjoyable and accessible for all users.
