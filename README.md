# CyberPath — Cybersecurity Career Discovery Quiz  
ICT171 Cloud Server Project — 2026  
Author: Satnaam Singh Rooprai

---

## 1. Introduction
CyberPath is an interactive, cloud‑hosted quiz designed to help beginners discover which cybersecurity career path best aligns with their personality, interests, and problem‑solving style.  
The project demonstrates practical cloud deployment, web development, documentation, and professional communication skills required in **ICT171 – Introduction to Server Environments and Architectures**.

---

## 2. Purpose of the Project
Cybersecurity is a broad field with many specializations, and newcomers often struggle to understand which path suits them.  
CyberPath addresses this challenge by providing:

- A guided **10‑question quiz**
- A personalized **career recommendation**
- Suggested **certifications** for beginners
- Curated **free learning resources**
- A modern, responsive interface built with HTML, CSS, and JavaScript

The goal is to make cybersecurity exploration simple, accessible, and tailored to each user.

---

## 3. What the Website Does
CyberPath evaluates how users think, solve problems, and approach challenges.  
Based on their responses, the system identifies traits such as:

- Vigilance  
- Curiosity  
- Precision  
- Aggression  
- Communication  
- Systems Thinking  

These traits are mapped to cybersecurity roles including:

- SOC Analyst  
- Penetration Tester  
- Cloud Security Specialist  
- Digital Forensics Analyst  
- Governance, Risk & Compliance (GRC)

Each result includes:

- A detailed role description  
- Key responsibilities  
- Recommended certifications  
- Free learning resources  
- A trait breakdown summary  

---

## 4. Problem the Project Solves
Cybersecurity is rapidly expanding, but beginners often feel overwhelmed by the number of career options.  
CyberPath solves this by:

- Reducing confusion  
- Providing clarity and direction  
- Offering structured learning pathways  
- Helping users understand their strengths  
- Encouraging exploration of cybersecurity roles  

This makes CyberPath valuable for students, career‑changers, and anyone entering cybersecurity.

## 5. Course Context (ICT171 Requirements)
This project fulfills the ICT171 Cloud Server Project requirements by demonstrating:

### ✔ Cloud Deployment (IaaS)
The website is hosted on a manually configured AWS EC2 Ubuntu server.

### ✔ Web Server Configuration
Apache2 serves the HTML/CSS/JS website.

### ✔ DNS Integration
A purchased domain is linked to the EC2 Elastic IP.

### ✔ SSL/TLS Implementation
The site is secured with HTTPS using a valid certificate.

### ✔ Scripting Component
JavaScript powers the quiz logic, scoring system, and dynamic results page.

### ✔ Professional Documentation
All setup steps and project details are documented clearly in GitHub.

### ✔ Video Explainer
A walkthrough video demonstrates the deployment process from start to finish.
